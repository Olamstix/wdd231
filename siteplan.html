MiBudget – Site Plan

A Personal Budgeting Web Application

Site Name

MiBudget

The name “MiBudget” was selected because it is simple, memorable, and emphasizes personal ownership (“My Budget”). It reflects the app’s goal of helping users manage and track their income and expenses efficiently.

Optional domain: mibudgetapp.com

Site Purpose

The MiBudget website serves as a personal finance management tool designed to help users record, organize, and analyze their financial activities. The platform will allow users to add income and expenses, view categorized transactions, and generate summaries that promote better budgeting habits.

Scenarios

  • How can I keep track of all my monthly income and expenses in one place?
  • Can I see a summary of my total earnings and spending to know if I’m saving or overspending?

Color Schema

The MiBudget color scheme emphasizes professionalism and trust, using cool blues and light backgrounds for clarity and calmness. The yellow accent color adds a friendly and modern touch.

  • Primary Color (#004aad): Used for headings, navigation bar, and logo.
  • Secondary Color (#f4f6fa): Used for page background and section contrast.
  • Accent Color (#ffb703): Used for buttons, highlights, and hover effects.
  • Text Color (#333333): Used for main body text.

Typography

The site will use clean, readable, and modern fonts that improve legibility and maintain a professional tone.

  • Headings: Montserrat, sans-serif
  • Body Text: Poppins, sans-serif
  • Accent/Buttons: Use bold Montserrat for emphasis

Testing and Validation

The MiBudget site will be tested using HTML and CSS validation tools, Lighthouse audits for performance and SEO, and color contrast checkers for accessibility. Regular testing will ensure compliance with best practices for responsive design, accessibility, and user experience.
